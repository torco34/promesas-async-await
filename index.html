typeof es un operador de JavaScript que devuelve una cadena que indica el tipo de dato de una expresión.

"undefined" si la expresión no tiene valor asignado
"boolean" si la expresión es de tipo booleano
"number" si la expresión es de tipo numérico
"string" si la expresión es de tipo cadena de texto
"object" si la expresión es de tipo objeto, incluyendo arrays y null "function" si la expresión es de tipo función "symbol" si la expresión es de tipo símbolo
